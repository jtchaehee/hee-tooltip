랑벨 반응형웹 구축 사례

  • CLIENT

    LANBELLE

  • RELEASE DATE

    2017.10

  • TYPE

    반응형웹, 다국어, CMS

프리미엄 천연 스킨케어인 랑벨의 리뉴얼 사이트는 건강한 아름다움을 전달하고자 하는 브랜드 컨셉을
가장 잘 표현할 수 있도록 제작하였습니다.
신규 브랜드로서 브랜드의 이미지를 잘 전달하고 브랜드의 철학을 잘 반영하는 구성과 구조로
브랜드 입지를 강화하는데 목적을 두었습니다.
랑벨 공식 브랜드 홈페이지는 웹어워드 2017 화장품분야 대상, 앤어워드 2017 위너, 디자인어워즈인아시아 DESIGN OF THE DAY, AWWWARDS HONORABLE MENTION을 수상하며 국내/해외 어워드 4관왕의 영예를 안았습니다.

사이트 바로가기

사용자 환경에 맞춰 최적화되는 반응형웹

다양한 디바이스와 해상도에 맞게 최적화되는 반응형웹으로 개발하여
여러 환경의 사용자에게 동일한 사용자경험을 제공하며, 사이트의 유지관리 효율성을 높였고
검색엔진에 친화적이게 제작되었습니다.

오직 랑벨만의 아이콘 디자인

콘텐츠를 쉽게 이해할 수 있도록 랑벨 심플퍼펙션을 반영한 일러스트아이콘을 제작하였습니다.

아름답고 유용하며 의미있는 디자인

메인은 고화질 비쥬얼 이미지를 사용한 풀 스크린 형태입니다. 풀 스크린 화면은
사이트의 아이덴티티를 폭발적으로 인지할 수 있게 해주며 복잡합 ui를 제거함과 동시에
긴 스크롤링 페이지로 넘어갈 분위기를 만들어 줍니다.
전체적인 톤&매너는 프리미엄 친환경 사이트로 내추럴한 느낌과 고급스러운 느낌을 가미하고자 하였습니다.
그리드를 깨고 비대칭을 활용한 레이아웃을 사용한 디자인은 기존의 화장품의 틀을 깬다는 의미로써
남다른 사고와 남다른 시각으로 새로운 화장품을 생각해내는 랑벨의 비전을 표현합니다.

404 에러페이지 디자인

브라우저 기본 404페이지는 지루하고 도움이 되지 않습니다.
맞춤 제작 404페이지로 에러페이지에 대한 방문자의 스트레스와 이탈률을 줄였습니다.
